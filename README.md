# Sistema de Cadastro em C#

Projeto desenvolvido para praticar lógica de programação utilizando C#.

# Funcionalidades

* Cadastro de pessoas
* Listagem de usuários
* Remoção com confirmação
* Edição de nomes
* Validação de entradas
* Prevenção de nomes duplicados

# Tecnologias

* C#
* .NET
* Console Application

# Objetivo

Praticar:

* Estruturas de repetição
* Condicionais
* Listas
* Validação de dados
* Manipulação de strings
